<h2 align='center'>Installation</h2>
1.Cloning a project<br>
    <code>git clone https://github.com/webtitovdev/obitski-wear.git</code><br>
2. Install <br>
   <code>npm instal</code><br>
3. Run <br>
   <code>npm start</code><br>
4. Build <br>
   <code>npm run build</code> <br>
